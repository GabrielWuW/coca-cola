O que é um repositório no Git e qual a sua importância?
  R: É um local onde é possível guardar projetos e acessar versões anteriores deles.

Qual a função do Git Bash?
  R: O Git Bash permite o gerenciamento de projetos Git através de comandos em um terminal.

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
  R: "git add <arquivo>" irá selecionar um arquivo para o commit, "git add ." seleciona todos os arquivos que estão na pasta.

Por que utilizamos o ponto (.) no comando git add .?
  R: Para selecionar todos os arquivos que estão na pasta.

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
  R: O comando git commit irá salvar as mudanças dos arquivos no repositório do projeto.

Qual a função do comando git push?
  R: git push envia o commit salvo no projeto local ao projeto remoto.

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
  R: Criar vários commits é importante pois eles ajudam a identificar erros mais rápido e permitem o retorno a versões anteriores do código.

Explique a função das tags HTML e como elas estruturam uma página.
  R: As tags HTML têm como objetivo organizar o código e isolar os elementos em caixas, facilitando o CSS. Elas estruturam seu conteúdo em blocos para o navegador exibi-los de forma correta.

O que são atributos em uma tag HTML e qual a sua função?
  R: Atributos são informações adicionais que podem ser colocadas junto das tags, sua principal função é associar algo àquela tag.

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
  R: O Emmet é uma tecnologia presente no VSCode que dá sugestões e completa a digitação de tags.

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
  R: O box-sizing define como é calculado a largura e altura de um elemento. Usamos "box-sizing: border-box" no reset para garantir que o tamanho da borda seja incluído no tamanho total do elemento.

Por que é importante utilizar tags semânticas no HTML?
  R: Tags semânticas são importantes pois facilitam a leitura do código e ajudam no processo da exibição do site na internet.

O que são media queries no CSS e qual a sua função?
  R: Os media queries têm a função de mudar o conteúdo de uma página conforme o tamanho da janela de exibição.

Qual a função do GitHub Pages?
  R: O GitHub Pages permite hospedar sites estáticos a partir de um repositório no GitHub.

O que é o Flexbox no CSS e para que ele é utilizado?
  R: O Flexbox facilita a distribuição e alinhamento de elementos dentro de um contêiner.

Qual a diferença entre justify-content e align-items em um container Flexbox?
  R: "justify-content" irá alinhar itens no eixo horizontal, "align-items" alinha no vertical. Essas configurações podem mudar conforme a orientação do contêiner definida no Flex.

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
  R: O "flex-grow" expande um item até ocupar todo o espaço disponível em seu contêiner pai, quanto maior o número, maior será sua prioridade de tamanho.

Explique a diferença entre flex-direction: column; e flex-direction: row;.
  R: "flex-direction: column" irá mudar a orientação dos itens de um contêiner para ficarem em colunas, já "flex-direction: row" deixará a orientação como linha.
